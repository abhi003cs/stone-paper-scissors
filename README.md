# 🎮 Rock Paper Scissors Game

An interactive Rock Paper Scissors game built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- Play against the computer
- Random computer moves
- Live score tracking
- Win/Lose/Draw message
- Reset Game button
- Responsive design
- Smooth hover effects

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
